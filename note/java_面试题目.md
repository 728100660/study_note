### 1、== 和 equals 的区别

​	== 在对基本数据类型，byte，char，short，int，long，float，double，Boolean比较时，是进行的值的比较，而在对复杂数据类型比较的时候，比较的是内存地址的值。

​	equals ：在java基类object中定义了一个方法equals（实际就是利用的== 方法），在没有重写equals的情况下，所有复杂类型的比较都是比较内存地址。

### 2、list 和 map 的区别

​	list 是一个有序容器，所有在list中的对象都是有序的，list运行插入重复数据和null元素

​	map：map中存放的数据是无须的，不重复的，以键值对形式存在，通过唯一的建来查找值，建唯一（键比较重复的规则按照equals比较），键值可重复，也可以是任意对象。
