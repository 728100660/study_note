连续记录的查询
    最初的问题是这样的:
我有一个表结构，
fphm,kshm
2014,00000001
2014,00000002
2014,00000003
2014,00000004
2014,00000005
2014,00000007
2014,00000008
2014,00000009
2013,00000120
2013,00000121
2013,00000122
2013,00000124
2013,00000125

(第二个字段内可能是连续的数据，可能存在断点。)

怎样能查询出来这样的结果,查询出连续的记录来。
就像下面的这样？
2014,00000001,00000005
2014,00000007,00000009
2013,00000120,00000122
2013,00000124,00000125

非常巧妙的答案:

SELECT b.fphm,
       MIN(b.kshm) start_hm,
       MAX(b.kshm) end_hm
  FROM (SELECT a.*,
               to_number(a.kshm - rownum) cc
          FROM (SELECT *
                  FROM t
                 ORDER BY fphm,
                          kshm) a) b
 GROUP BY b.fphm,
          b.cc

FPHM START_HM END_HM
---------- -------- --------
2013 00000120 00000122
2013 00000124 00000125
2014 00000001 00000005
2014 00000007 00000009
方法总结：数字－ROWNUM进行分组即可
